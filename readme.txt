Boot Dashboard is a custom bootstrap wordpress admin style with dark version.
 
== Description ==
 
It's an awosome ui/ux wordpress dashboard template. Easy to install and use. After install the plugin your wordpress default dashboard style change and get awosome, professional look.
 
== Features ==  

* Custom Dashboard
* Bootstrap Dashboard
* Responsive Design
* Professional Color
* UI/UX Outlook
* Simple Use
 
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
  
1. Upload `boot-dashboard` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Now just look the change !
